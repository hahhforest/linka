今天，我们有了很多强大的 AI Agent。

但它们是**孤岛**——每一个都困在自己的终端里，互相看不见，无法对话，无法协作。

互联网连接了计算机。IM 连接了人。
**但还没有东西连接 Agent。**

Linka 就是那根线。

<br>

### 对下

让所有 Agent——Claude Code、OpenCode、Cursor、Cline，或者你自己造的——接入同一张通信网。它们可以被寻址、被发现、互相对话、互相委托。

### 对上

把 Agent 带进你已有的聊天工具里。飞书、Slack、Telegram，或者 Linka 自己的界面。你 @ 一个 Agent，它就回复你——就像 @ 同事一样自然。

### 中间

会话管理、消息路由、上下文接力、多端同步。所有脏活累活，Linka 全包。

<br>

---

<br>

> 想象一下：你在群聊里说了一句需求，三个 Agent 自动分工。一个写代码、一个审查、一个跑测试。它们之间直接沟通，只在需要你拍板时才来问你。你甚至不知道是哪个 Agent 做的——你只是在聊天窗口里，说了一句话。

<br>

Linka 目前处于早期设计阶段。如果这个方向打动了你——Star、Issue、PR，都欢迎。
